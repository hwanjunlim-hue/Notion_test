# 대시보드 차트 무한 로딩 버그 수정

## 문제 분석

차트 컴포넌트가 데이터 페칭 요청을 완료하지 못하거나, 데이터 로드 상태가 제대로 업데이트되지 않아 무한 스피너 상태에 머물러 있는 문제입니다.

## 수정 방안

### 1단계: 데이터 페칭 로직 검증

```javascript
// dashboard/ChartComponent.jsx - 수정 전
useEffect(() => {
  fetchChartData();
}, []); // 의존성 배열 누락으로 인한 무한 재호출
```

```javascript
// dashboard/ChartComponent.jsx - 수정 후
useEffect(() => {
  let isMounted = true;
  
  const loadData = async () => {
    try {
      setLoading(true);
      const response = await fetchChartData();
      if (isMounted) {
        setData(response);
        setError(null);
      }
    } catch (err) {
      if (isMounted) {
        setError(err.message);
      }
    } finally {
      if (isMounted) {
        setLoading(false);
      }
    }
  };
  
  loadData();
  
  return () => {
    isMounted = false; // 언마운트 시 상태 업데이트 방지
  };
}, []); // 올바른 의존성 배열
```

### 2단계: 타임아웃 설정

```javascript
// api/chartService.js
const TIMEOUT_MS = 10000; // 10초 타임아웃

export const fetchChartData = async (config = {}) => {
  const controller = new AbortController();
  const timeoutId = setTimeout(() => controller.abort(), TIMEOUT_MS);
  
  try {
    const response = await fetch('/api/chart-data', {
      signal: controller.signal,
      ...config
    });
    clearTimeout(timeoutId);
    
    if (!response.ok) {
      throw new Error(`HTTP Error: ${response.status}`);
    }
    return await response.json();
  } catch (error) {
    clearTimeout(timeoutId);
    throw error;
  }
};
```

### 3단계: 에러 핸들링 UI 추가

```javascript
// dashboard/ChartComponent.jsx
return (
  <div className="chart-container">
    {loading && <Spinner />}
    {error && (
      <ErrorBoundary 
        message={error}
        onRetry={() => loadData()}
      />
    )}
    {data && <Chart data={data} />}
  </div>
);
```

### 4단계: API 응답 검증

```javascript
// api/chartService.js - 응답 검증 추가
export const fetchChartData = async () => {
  const response = await fetch('/api/chart-data');
  const json = await response.json();
  
  // 응답 데이터 구조 검증
  if (!json || typeof json !== 'object' || !Array.isArray(json.data)) {
    throw new Error('Invalid data format received');
  }
  
  return json;
};
```
