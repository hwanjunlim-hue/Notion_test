# 모바일에서 드롭다운 메뉴 터치 영역 겹침

## 모바일 드롭다운 메뉴 터치 영역 겹침 수정

### 1단계: CSS 터치 영역 최적화
- 드롭다운 메뉴 항목 최소 높이를 44px 이상으로 설정 (모바일 터치 표준)
- 각 메뉴 항목 간 명확한 padding 추가 (상하 12px 이상)
- z-index 값 충돌 확인 및 정렬

### 2단계: HTML 구조 검토
- 각 드롭다운 항목이 독립적인 `<button>` 또는 `<a>` 태그로 래핑
- 터치 이벤트 캡처를 위한 `touch-action: manipulation` 속성 추가

### 3단계: JavaScript 이벤트 처리
- 터치 이벤트와 클릭 이벤트 분리 처리
- 버블링 방지 (`stopPropagation()`) 적용
- 더블 탭 줌 비활성화 (user-scalable=no 대신 touch-action 사용)

### 4단계: 뷰포트 메타 태그 확인
html
<meta name="viewport" content="width=device-width, initial-scale=1.0">


### 5단계: 미디어 쿼리 적용
css
@media (max-width: 375px) {
  .dropdown-item {
    min-height: 48px;
    padding: 14px 16px;
    touch-action: manipulation;
  }
}
