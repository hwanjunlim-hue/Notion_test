# 모바일에서 드롭다운 메뉴가 화면 밖으로 잘림

## 모바일 드롭다운 메뉴 잘림 현상 수정

### 1단계: CSS 뷰포트 제약 조건 확인
- `overflow: hidden`이 부모 요소에 적용되어 있는지 확인
- 드롭다운 메뉴 부모 요소의 `overflow` 속성을 `visible`로 변경

### 2단계: 드롭다운 위치 조정
css
.dropdown-menu {
  position: absolute;
  left: auto;
  right: 0;  /* 오른쪽 정렬 */
  white-space: nowrap;
}


### 3단계: 모바일 전용 미디어 쿼리 추가
css
@media (max-width: 768px) {
  .dropdown-menu {
    position: fixed;
    max-width: 90vw;
    right: 5vw;
    left: auto;
    z-index: 1000;
  }
}


### 4단계: 컨테이너 제약 조건 해제
- 부모 네비게이션 요소에 `overflow: visible` 설정
- 필요한 경우 `z-index` 값 증가 (최소 1000)

### 5단계: JavaScript 위치 자동 계산 (선택)
- 드롭다운 오픈 시 화면 오른쪽 여백 체크
- 잘릴 경우 자동으로 위치 조정하는 로직 추가
