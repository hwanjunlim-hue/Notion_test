# 대시보드 차트 로딩 시 10초 이상 지연

## 대시보드 차트 로딩 지연 문제 해결

### 1단계: 데이터 캐싱 구현
- Redis 또는 메모리 캐시를 사용하여 차트 데이터 캐싱
- 캐시 만료 시간 설정 (예: 5분)
- 데이터 갱신 시에만 캐시 무효화

### 2단계: API 최적화
- 불필요한 데이터 필드 제거 (필요한 필드만 SELECT)
- 데이터베이스 인덱스 추가 (날짜, 매출액 컬럼)
- 페이지네이션 또는 데이터 범위 제한 (최근 12개월만 조회)

### 3단계: 프론트엔드 성능 개선
- 가상 스크롤링 적용
- 청크 로딩으로 초기 데이터만 먼저 표시
- 로딩 상태 UI 개선 (스켈레톤 로더 추가)

### 4단계: 비동기 처리
- 차트 렌더링을 Web Worker에서 처리
- 메인 스레드 블로킹 제거
- requestIdleCallback을 사용한 우선순위 처리

### 5단계: 데이터 압축
- gzip 압축 활성화
- 불필요한 소수점 제거
- 데이터 포맷 최적화
