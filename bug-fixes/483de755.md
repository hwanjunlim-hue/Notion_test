# 대시보드 차트 로딩 시 5초 이상 지연 발생

## 대시보드 차트 로딩 지연 수정 방안

### 1단계: 데이터 쿼리 최적화
- 차트 데이터 조회 쿼리에 인덱스 추가 (날짜, 카테고리 필드)
- 불필요한 JOIN 제거 및 SELECT 필드 최소화
- 데이터베이스 쿼리 실행 계획 분석 및 개선

### 2단계: 캐싱 메커니즘 구현
- Redis 캐싱 도입 (매출 데이터 1시간 TTL)
- 캐시 무효화 전략 수립 (데이터 업데이트 시점)
- 캐시 히트율 모니터링

### 3단계: 프론트엔드 성능 최적화
- 차트 라이브러리 라이징 (recharts → lightweight 대안)
- 데이터 페이지네이션 구현 (초기 로드 데이터 50% 감소)
- 가상 스크롤링 적용

### 4단계: 백엔드 비동기 처리
- API 응답 압축 (gzip) 활성화
- 데이터 집계를 별도 배경 작업으로 처리
- 부분 로딩 구현 (프로그레시브 렌더링)

### 5단계: 모니터링 및 로깅
- 차트 로딩 시간 메트릭 수집
- 쿼리 실행 시간 로깅
- APM 도구 통합 (NewRelic, DataDog 등)
